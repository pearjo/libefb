LibEFB Python binding
=====================

Python binding to the Electronic Flight Bag library LibEFB. To get started
checkout the :ref:`user guide <userguide>`. For more on all available types
refer to the :ref:`API reference <reference>`. To get more background and
detailed information, checkout the documentation of the ``efb`` Rust library.
